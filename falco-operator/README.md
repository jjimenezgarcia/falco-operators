# Falco operator

A [Juju](https://juju.is/) charm deploying and managing [Falco](https://falco.org/), an open-source
runtime security tool for physical and virtual machines. This subordinate charm simplifies the
deployment, configuration, and lifecycle management of Falco instances.


## Release

Falco operator uses the [falco binary](https://github.com/falcosecurity/falco/tree/master) and [falco plugins](https://github.com/falcosecurity/plugins) directly built from source. Please check the [RELEASE.md](./RELEASE.md) to learn more about the process.
